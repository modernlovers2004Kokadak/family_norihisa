# 則久家 家系図

GitHub Pages でそのまま公開できる完成版です。

## 使い方

1. このフォルダ内の `index.html` と `README.md` を GitHub リポジトリへアップロードします。
2. GitHub Pages を有効にします。
3. `index.html` がトップページとして表示されます。

## 内容

- 則久家 家系図
- 人物詳細 一覧
- 戸籍記載 原文
- スマホ閲覧対応
- 印刷対応
- 人物詳細では旧姓を名前の上に小さく表示


更新: 林竹治・則久紋治・則久虎蔵の実親情報を反映する方針。
