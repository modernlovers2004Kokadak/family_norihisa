# 則久家 家系図

GitHub Pagesで公開できる完成版です。

## 使い方

1. このフォルダ内のファイルを、GitHubリポジトリのルートにアップロードします。
2. `Settings` → `Pages` → `Branch` を `main`、フォルダを `/(root)` に設定します。
3. 数分後に公開ページで確認します。

## ファイル構成

- `index.html`：家系図本体
- `style.css`：画面表示・印刷用デザイン
- `script.js`：人物カードから人物詳細へ移動する補助処理
- `.nojekyll`：GitHub Pages用設定

## 備考

戸籍資料をもとに、家系図と人物詳細一覧を作成しています。
印刷時も家系図と人物詳細一覧が出力されます。
